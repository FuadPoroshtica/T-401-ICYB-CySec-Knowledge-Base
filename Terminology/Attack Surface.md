# Attack Surface
**Term**: Attack Surface

**Definition**: The complete set of ways an attacker can interact with or send input to a system in order to try to compromise it.

**Context/Example**: Email, USB sticks, Wi-Fi, Bluetooth, and downloaded software are all parts of the attack surface.
