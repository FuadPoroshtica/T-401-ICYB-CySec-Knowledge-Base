# TCP IP Model

The TCP/IP (Transmission Control Protocol/Internet Protocol) model is another widely used networking model, especially for the internet. It only uses these four layers:

1. Link Layer (combines OSI Layers [1-Physical Layer](../OSI Model/1-Physical Layer.md) and [2-Data Link Layer](../OSI Model/2-Data Link Layer.md))
2. Internet Layer (OSI Layer [3-Network Layer](../OSI Model/3-Network Layer.md))
3. Transport Layer (OSI Layer [4-Transport Layer](../OSI Model/4-Transport Layer.md))
4. Application Layer (combines OSI Layers [5-Session Layer](../OSI Model/5-Session Layer.md), [6-Presentation Layer](../OSI Model/6-Presentation Layer.md), and [7-Application Layer](../OSI Model/7-Application Layer.md))

Great lies in networking: “An application only needs to understand the Application Layer to communicate over a TCP/IP network.”
