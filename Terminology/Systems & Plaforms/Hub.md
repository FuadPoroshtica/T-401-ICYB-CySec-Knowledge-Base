# Hub
**Term**: Network Hub

**Definition**: Also called an “Ethernet Hub”, used to connect multiple Ethernet devices together and making them act as a single network segment. If a signal is induced at any input, every output is open, except the original.

**Context/Example**: Often used in LANs, for quicker connection.

**Related Concepts**: Network, Switch, [[Router]]
