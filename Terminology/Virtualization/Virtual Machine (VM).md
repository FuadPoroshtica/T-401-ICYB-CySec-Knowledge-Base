# Virtual Machine (VM)
**Term**: Virtual Machine (VM) / Guest Machine / Guest OS

**Definition**: A software-defined computer that runs inside the host, with virtual CPU, RAM, disk, and network; it has its own [Operating System](../Systems%20&%20Plaforms/Operating%20System.md).

**Context/Example**: A Kali Linux VM running inside VirtualBox is a guest [OS](../Systems%20&%20Plaforms/Operating%20System.md) / VM. It *acts* like its own machine, but itâ€™s just software.
