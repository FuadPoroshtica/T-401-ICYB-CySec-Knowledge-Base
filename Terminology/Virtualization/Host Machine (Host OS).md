# Host Machine (Host OS)

**Term**: **Host Machine / Host OS**

**Definition**: The *real* physical computer and its [Operating System](../Systems & Plaforms/Operating System.md), on top of which [Virtualization](Virtualization.md) software runs.

**Context/Example**: A laptop running Windows or Linux and hosting VirtualBox is the “host”; the [VMs](Virtual Machine (VM).md) inside VirtualBox are the “guests”.

**Related Concepts**: [Guest OS](Virtual Machine (VM).md), [hypervisor](Hypervisor.md), physical hardware.
