# Host Machine (Host OS)

**Term**: **Host Machine / Host OS**

**Definition**: The *real* physical computer and its [Operating System](../Systems%20&%20Plaforms/Operating%20System.md), on top of which [Virtualization](Virtualization.md) software runs.

**Context/Example**: A laptop running Windows or Linux and hosting VirtualBox is the “host”; the [VMs](Virtual%20Machine%20(VM).md) inside VirtualBox are the “guests”.

**Related Concepts**: [Guest OS](Virtual%20Machine%20(VM).md), [hypervisor](Hypervisor.md), physical hardware.
