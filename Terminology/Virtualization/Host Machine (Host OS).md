# Host Machine (Host OS)

**Term**: **Host Machine / Host OS**

**Definition**: The *real* physical computer and its operating system, on top of which virtualization software runs.

**Context/Example**: A laptop running Windows or Linux and hosting VirtualBox is the “host”; the VMs inside VirtualBox are the “guests”.

**Related Concepts**: [Guest OS](Virtual%20Machine%20(VM).md), [hypervisor](Hypervisor.md), physical hardware.
