# PWD (Phishing Website Detection)
**Term**: PWD (Phishing Website Detection)

**Definition**: PWD (Phishing Website Detection) refers to techniques and tools used to identify and block phishing websites that attempt to steal sensitive information from users by masquerading as legitimate sites.

**Context/Example**: Many web browsers incorporate PWD features to warn users when they attempt to visit known phishing sites.

**Related Concepts**: Phishing, Cybersecurity, Web Security
