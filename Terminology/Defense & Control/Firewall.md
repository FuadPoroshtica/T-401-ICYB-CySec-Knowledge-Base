# Firewall
**Term**: Firewall

**Definition**: A security device or software that filters network traffic based on rules, deciding which connections are allowed or blocked. This cannot actually read the incoming packets however, if you were to unknowingly download a virus from a website, there is nothing a firewall can do to stop you.
